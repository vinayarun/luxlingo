# Benchmark: LuxLingo vs. Llo.lu

## 📊 Overview
**Llo.lu** is the official state-supported platform for learning Luxembourgish. It is comprehensive but suffers from usability and accessibility issues typical of institutional software. **LuxLingo** aims to be the agile, user-centric alternative.

## ⚔️ Feature Comparison

| Feature | Llo.lu (Official App) | LuxLingo (Our App) |
| :--- | :--- | :--- |
| **Target Audience** | General (Multilingual background) | **English Speakers** (Specific) |
| **Instruction Language** | **Luxembourgish Only** (Immersion) | **English** (Guided) |
| **Access** | Account Required (Complex registration) | **No Account** (Instant start) |
| **Connectivity** | Online Dependent (Video streaming) | **Offline First** (All assets local) |
| **UI/UX** | "Messy", "Cluttered", Institutional | **Simple**, Premium, Gamified |
| **Content Style** | Academic, Video-heavy | **Interactive**, Exercise-heavy |
| **Pacing** | Steep (Grammar heavy early on) | **Gradual** (i+1, Gamified repetition) |

## 🧩 Curriculum Gap Analysis

### Llo.lu A1 Curriculum
*   **Strengths**: Comprehensive, covers "Official" needs (Bank, Post Office, Administration).
*   **Weaknesses**:
    *   **Immersion Shock**: Teaches grammar *in* Luxembourgish, which confuses absolute beginners.
    *   **Cognitive Load**: Introduces complex vocabulary (professions, countries) very early.

### LuxLingo Opportunity
1.  **The "English Bridge"**: Llo.lu users complain about the lack of English explanations. We explicitly bridge this gap.
2.  **Micro-Learning**: Llo.lu lessons can be long. We focus on 5-10 minute bursts suitable for commuting.
3.  **Survival First**: We prioritize "ordering a beer" and "buying a ticket" over "naming countries" or "bank administration".

## 🚀 Key Differentiators to Double Down On
1.  **Zero Friction**: The "No Account" policy is a massive advantage over Llo.lu's complex registration.
2.  **Speed**: Llo.lu is reported to be slow/buggy. We must ensure LuxLingo remains instant and snappy.
3.  **English Context**: Every new word in LuxLingo is contextualized in English, solving the #1 complaint about Llo.lu.
